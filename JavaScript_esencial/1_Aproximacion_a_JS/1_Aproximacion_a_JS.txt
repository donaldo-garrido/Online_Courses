1.1 Qué es JavaScript
Es el lengiaje de programación que permitirá crear aplicaciones web y móviles
JS se encuentra en cualquier web

El tridente de la web es HTMS, CSS y JS
HTML es solo de marcado
CSS se enfoca en cómo se verá el contenido (presentación)
JS lenguaje interpretado que interactúa con HTML y CSS. Manipula el contenido. 
Invoca datos y servicios remotos.

JS fue creado en 1995 (No confundir JavaScrip con Java)
